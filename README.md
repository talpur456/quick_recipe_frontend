# Quick Recipe App 
A simple web app that helps you find recipes based on the ingredients you have at home. Enter your ingredients, see recipe suggestions, and view detailed recipe information including ingredients and instructions.

## Technologies Used
- HTML, CSS, JavaScript
- Masonry.js & images Loaded for the layout
- Minor AI assistance used for guidance and code suggestions

## Features
- Search recipes by ingredients
- View recipe details (ingredients, instructions, images)
- Responsive design and Masonry layout for recipe cards
- Easy-to-use interface

## Installation and Setup
### Clone the repository
git clone https://github.com/talpur456/quick_recipe_frontend.git
cd quick_recipe_frontend

### Backend Requirement
This frontend requires a backend server that provides recipe data. By default, it expects the backend API to be running and accessible at Render

### Run the app
- Open index.html in your browser
- No additional dependencies are needed for the frontend

## Usage
1. Enter ingredients in the search box (comma-separated)
2. Click Find Recipes. The frontend will fetch recipes from the backend server
3. Click Prepare on a recipe card to view recipe details (ingredients, instructions, image)
